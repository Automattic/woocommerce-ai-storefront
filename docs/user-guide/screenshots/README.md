## Re-capturing

If you re-style the admin UI or upgrade WooCommerce in a way that changes the chrome, refresh these images. The originals were generated by html2canvas inside a Claude-driven Chrome session — the same approach reproduces 1:1 with no theme dependency.

If you need to populate stats for shots `10` / `11` / `12` on a fresh install (no real AI orders yet), stamp a test order with the right meta via WP-CLI:

```bash
wp post meta update <order_id> _wc_order_attribution_utm_medium ai_agent
wp post meta update <order_id> _wc_order_attribution_utm_source chatgpt
```

Then re-save the order to trigger AI Storefront's attribution capture.
